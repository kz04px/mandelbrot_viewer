Mandelbrot Viewer

Using:
- OpenGL 4
- GLFW

Controls:
- Page Up: Increase iterations by 50
- Page Down: Decrease iterations by 50
- Plus key: Increase iterations by 1
- Minus key: Decrease iterations by 1
- R: Reset view
- T: Toggle trippy mode
- S: Print details
- F2: Take screenshot
- F12: Toggle fullscreen mode
- Esc: Close

Platforms:
- Windows
- Linux
